# XAI Pneumonia Demo

This project demonstrates explainable AI (XAI) for pneumonia detection using Grad-CAM and SHAP on three models:
- Baseline CNN
- Pretrained VGG16
- Random Forest

## Features
- Upload an image for pneumonia prediction
- View Grad-CAM and SHAP explanations
